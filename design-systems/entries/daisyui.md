# daisyUI

Tailwind plugin that layers semantic class names (`btn`, `btn-primary`, `card`, `modal`) and a theme system on top of Tailwind.

- **When to pick**: you want Tailwind + ready-made theme variants without framework lock-in.
- **Framework**: works with React, Vue, Svelte, static HTML.
