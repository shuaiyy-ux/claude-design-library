# Radix UI

Low-level, unstyled, accessible React primitives. The foundation under shadcn/ui.

- **Primitives**: Dialog, Popover, DropdownMenu, Tabs, Tooltip, Select, Toast, and more.
- **Styling**: none by default — pair with Tailwind, vanilla CSS, or a styling engine.
- **When to pick**: you need rock-solid a11y + keyboard behavior but want full visual control.
