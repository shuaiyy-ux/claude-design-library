# Headless UI

Tailwind Labs' unstyled, a11y-focused components for React and Vue. Smaller scope than Radix but well-integrated with Tailwind.

- **Components**: Dialog, Menu, Listbox, Combobox, Switch, Tabs, Disclosure, Transition.
- **When to pick**: Tailwind shop that wants official Tailwind-team bindings.
