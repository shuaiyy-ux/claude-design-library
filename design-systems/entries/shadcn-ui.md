# shadcn/ui

Not a library — a **collection of components** you copy into your codebase. Built on Radix primitives and Tailwind. Full source ownership.

- **Theming**: CSS variables (`--background`, `--foreground`, etc.) + light/dark.
- **Distribution**: `npx shadcn@latest add button`.
- **When to pick**: you want full control over component source and idiomatic Tailwind.
