# Material UI

React implementation of Google's Material Design. Huge ecosystem — MUI Core, Joy UI, MUI X (data grid, date pickers).

- **When to pick**: you want Material Design, or need MUI X's data grid.
- **Bundle**: larger than most — consider MUI Base (unstyled) for lighter builds.
