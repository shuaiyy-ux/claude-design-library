# Skeuomorphic

UI elements that imitate physical materials. Pre-flat iOS (iOS 6 and earlier) and OS X defined the web's version.

## Key rules
- **Gradients**: every surface has a subtle top-down light gradient.
- **Textures**: linen, felt, leather, brushed metal as backgrounds.
- **Bevels**: 1px inner highlight on top, 1px inner shadow on bottom.
- **Outer shadow**: soft, small, to detach from surface.
- **Icons**: photorealistic, not flat.

## When to use (rare)
- Niche nostalgia products, music apps, calculator demos.
- Mostly a historical reference now — but tokens are useful for novelty pages.
