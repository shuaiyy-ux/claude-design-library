# Editorial Magazine

Inspired by long-form magazines (Kinfolk, Cereal, The New Yorker online). Print sensibility on the web.

## Key rules
- **Type**: serif display (Playfair, Canela, GT Sectra) + serif body (Lora, Source Serif).
- **Drop caps** on leading paragraph.
- **Columns**: 1–2 column text, max-width ~680px.
- **Color**: cream or warm white background, near-black text, occasional sepia accent.
- **Rules**: thin horizontal rules between sections.
- **Imagery**: full-bleed photos with italic small-caps captions.
