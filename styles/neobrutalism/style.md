# Neo-Brutalism

Brutalism's cheerful cousin. Keeps the hard borders and offset shadows but swaps saturation for pastels and adds small rounded corners.

## Key rules
- **Borders**: 2–3px solid black.
- **Corners**: 4–8px — softened, not sharp.
- **Shadows**: hard offset (`6px 6px 0 0 #000`).
- **Color**: pastel palette (mint `#B5EAD7`, peach `#FFB5A7`, periwinkle `#B5B5FF`, butter `#FFE066`).
- **Type**: geometric sans (Inter, Space Grotesk, Satoshi).
- **Mood**: friendly, confident, web3/indie-SaaS adjacent.
