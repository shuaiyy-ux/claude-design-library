# Claymorphism

UI elements that look molded out of clay. Chunky, rounded, with double inset shadows simulating dimension.

## Key rules
- **Radius**: 20–32px. Never sharp.
- **Shadow**: **two inset shadows** — light from top-left, dark from bottom-right — plus an outer drop shadow.
- **Fill**: soft pastel or near-white.
- **Spacing**: generous padding (24–40px).
- **Type**: rounded sans (Nunito, Quicksand, Poppins).
