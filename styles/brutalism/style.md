# Brutalism

Confrontational, anti-polished. Treats the browser like raw concrete: visible structure, no decoration, no easing.

## Key rules
- **Borders**: 2–4px solid black on almost everything.
- **Corners**: sharp (`border-radius: 0`).
- **Shadows**: offset hard shadows (`8px 8px 0 0 #000`), never blurred.
- **Color**: one saturated primary (yellow, magenta, electric blue) + black on white.
- **Type**: system sans or mono. Mix sizes aggressively; let text overflow.
- **Motion**: none, or step-function only.

## Anti-patterns
- Drop shadows with blur
- Gradients
- Soft gray UI chrome
- "Tasteful" type pairings
